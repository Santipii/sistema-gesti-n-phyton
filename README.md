# Sistema de gestión en Python

Proyecto desarrollado como parte del curso de programación en equipo.

## Descripción
Aplicación en Python orientada a la gestión de entidades (personas, médicos, socios, etc.), permitiendo organizar y manipular datos mediante consola.

## Estructura
- entities/: clases del sistema
- exceptions/: manejo de errores
- main.py: punto de entrada

## Tecnologías
- Python
